# Fern and Petals Sales Analysis(Interactive Dashboard creation using MS Excel)
## Objectives
- Analyze overall sales and revenue performance
- Identify top-performing products and categories
- Understand customer purchasing patterns
- Track delivery performance
- Discover seasonal and occasion-based trends

## Key Metrics
- Total Orders: 1000,
- Total Revenue: ₹35,20,984,
- Average Customer Spend: ₹3,520.98,
- Average Delivery Time: 5.53 days,

## Dashboard Insights
- Occasion-Based Sales: Highest revenue from Anniversary and Raksha Bandhan
- Category Performance: Colors leads, followed by Soft Toys and Sweets
- Time Analysis: Peak sales during evening hours
- Monthly Trends: Revenue spikes in March and August
- Top Products: Magnam Set and Quia Gift are top contributors
- Top Cities: High orders from Imphal, Dhanbad, and Kavali

## Key Learnings
- Built interactive dashboards using Excel
- Performed data cleaning and transformation
- Created dynamic reports using Pivot Tables
- Improved data visualization and analytical skills
